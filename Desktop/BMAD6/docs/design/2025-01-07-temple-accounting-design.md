## üîå API Endpoints

### Authentication
```
POST /api/auth/register
  Body: { username, password, email, temple_name }
  Response: { id, username, temple_name }

POST /api/auth/login
  Body: { username, password }
  Response: { token, user }

POST /api/auth/logout
  Response: { message: "Logged out" }
```

### Transactions
```
GET /api/transactions?month=2025-01&category=&type=
  Response: [ { id, type, category, amount, date, description, details } ]

POST /api/transactions
  Body: { type, category_id, amount, date, description, details }
  Response: { id, ... }

PUT /api/transactions/:id
  Body: { type, category_id, amount, date, description, details }
  Response: { id, ... }

DELETE /api/transactions/:id
  Response: { message: "Deleted" }
```

### Categories
```
GET /api/categories
  Response: [ { id, name, type, is_default } ]

POST /api/categories
  Body: { name, type }
  Response: { id, name, type }

PUT /api/categories/:id
  Body: { name }
  Response: { id, ... }

DELETE /api/categories/:id
  Response: { message: "Deleted" }
```

### Reports
```
GET /api/reports/summary?month=2025-01&format=summary
  Response: { total_income, total_expense, balance, by_category }

GET /api/reports/export?month=2025-01&format=pdf|excel
  Response: (File download)
```

### Notifications
```
GET /api/notifications
  Response: [ { id, message, type, is_read, created_at } ]

PUT /api/notifications/:id
  Body: { is_read: true }
  Response: { id, ... }
```
